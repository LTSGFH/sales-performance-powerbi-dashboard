# Sales Performance Dashboard (Power BI)

## Project Overview
This project analyzes customer insights, sales performance, and sales team performance using Power BI.  
The dashboard helps track revenue trends, customer distribution, and sales target achievement.

## Tools Used
- Power BI
- Excel
- DAX
- Data Modeling

## Dashboard Pages

### 1. Customer Insights
- Total customers
- Gender distribution
- Customer age segmentation
- Country-wise customers

### 2. Sales Performance
- Total revenue
- Total orders
- Monthly order trends
- Highest order value
- Orders by country (Map)

### 3. Sales Team Performance
- Target vs Achieved sales
- Average shortfall analysis
- Best team of the year
- Best manager by country

## Key Insights
- USA has the largest customer base.
- Total revenue reached 12.28M across 2500 orders.
- Team Alpha achieved the highest sales performance.
